# Оношлогоо

Next.js + TypeScript фронтенд үйлдлийн систем. Энэ нь шинж тэмдгүүд болон текстэр тайлбараас локал оношилгооны магадлалыг шууд render хийж харуулна.

## Яаж ажиллуулах вэ

### Алхам 1: Тусгалын хавтас руу оч
```bash
node -v
```

### Алхам 2: Node болон npm байгаа эсэхийг шалгах
```bash
node -v
npm -v
```

- Хэрвээ `node: command not found` эсвэл `npm: command not found` гэж гарвал Node.js суулгах хэрэгтэй.

### Алхам 3: Node.js суулгах
- Хэрвээ Homebrew байгаа бол:
```bash
brew install node
```

- Эсвэл https://nodejs.org/ -оос LTS хувилбарыг татаж суулгана.

### Алхам 4: Хамаарал суулгах
```bash
npm install
```

### Алхам 5: Хөгжүүлэлтийн сервер стартлах
```bash
npm run dev
```

### Алхам 6: Хөтөч нээх
```text
http://localhost:3000
```

## Файлын бүтэц

- `pages/index.tsx` - үндсэн нүүр хуудас, шинжилгээний UI болон локал оношилгооны логик.
- `styles/globals.css` - дэлхийн стилинг.
- `package.json` - Next.js, React, TypeScript хамаарал.

## Тайлбар

- Энэ сайт нь бодит эмнэлгийн онош биш.
- Хэрэглэгчийн оруулсан шинж тэмдэг дээр суурилан боломжит нөхцлүүдийг харуулна.
- `next dev` команд Next.js хөгжүүлэлтийн серверийг ажиллуулна.
